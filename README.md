## ✅ 주요 기능
1. CRUD 기능 : 멤버카드 읽기, 방명록 등록하기

2. 오픈API를 이용해 fetch로 데이터 가져와서 nav_bar에 실시간 날씨정보 출력

3. 멤버카드에서 사진을 클릭 시 개인 상세 프로필을 모달창을 생성하여 띄움

4. firebase를 활용하여 firestore DB에 방명록 입력 내용을 저장하고 데이터 불러옴
